*DATASCI 306 (Statistical Programming) - UMich*

Led a team of five for a comprehensive statistical analysis and forecasting project using R, addressing global population trends using the UN World Population Prospects dataset.

Implemented a **Prophet machine learning model** with external regressors to forecast population dynamics through 2100. Created **interactive visualizations via Shiny app**.

*Final Grade: 26/25 (With 1 extra credit). Selected as one of three excellent projects out of ~50 groups.*
